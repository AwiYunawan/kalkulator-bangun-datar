Keliling Segitiga

Rumus Keliling Segitiga

K = S1 + S2 + S3


Dimana

K = Keliling

S1 = sisi AB

S2 = sisi BC

S3 = sisi CA

Luas Jajargenjang

Rumus Luas Jajargenjang

L = a x t


Dimana

L = Luas

a = Alas

t = Tinggi

Hitung Luas Jajargenjang


Keliling Jajargenjang

Rumus Keliling Jajargenjang

K = 2 x ( a + b )


Dimana

K = Keliling

a = sisi a (alas)

b = sisi b (miring)
